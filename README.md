# forecastUNAL

El objetivo de forecastUNAL es facilitar la implementación de modelos y análisis de series de tiempo de la asignatura Estadística III (3009137) del Departamento de Estadística de la Universidad Nacional de Colombia, Sede Medellín.

## Installation

Usted puede instalar la versión de desarrollo de `forecastUNAL` de la siguiente manera:

``` r
#install.packages("devtools")
devtools::install_github("cmlopera/forecastUNAL")
```
